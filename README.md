# Student GPA Analysis Dashboard

This project analyzes student GPA data using Python and presents insights through an interactive Streamlit dashboard.

## Features
- Student GPA dataset display
- Summary statistics
- Semester-wise GPA analysis
- Top & bottom performers
- GPA distribution histogram

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
